# CAN ID Hunter v1.2 — ESP32 CYD

Поиск CAN ID подсветки приборки автомобиля. Только сканер, без управления LED.

## Изменения в v1.2 (обновлено 27.05.2026)

### Компоновка MONITOR (новая)
- **Бар слева** (60×170, с скруглённой рамкой), **% крупно** (textSize 5), **raw value** (textSize 3) под %
- **Анти-мерцание**: лимитер обновления 50ms — экран перерисовывается не чаще 20 раз/сек, а не 100 раз/сек
- `updateMonitorValue()` обновляет **только бар, процент и raw** — шапка, рамка, подвал не трогаются
- `redrawNeeded` убран из MONITOR — полная перерисовка только при смене режима
- displayRawVal (0-255) и brightnessVal (%) разделены

## Подключение MCP2515 (требуется пайка)

| MCP2515 | CYD сигнал | Куда на CYD | Пайка? |
|---------|-----------|-------------|--------|
| VCC (5V) | 5V | Площадка **S3** сзади платы | ⚠️ Да |
| GND | GND | Площадка **S1** сзади платы | ⚠️ Да |
| SCK | GPIO 18 | Ножка ESP32 | ⚠️ Да |
| MOSI | GPIO 23 | Ножка ESP32 | ⚠️ Да |
| MISO | GPIO 19 | Ножка ESP32 | ⚠️ Да |
| **CS** | **GPIO 22** | **Разъём P3, пин 3** | ❌ **Нет** |
| CAN_H | — | Клемма MCP2515 → OBD2 пин 6 | ❌ |
| CAN_L | — | Клемма MCP2515 → OBD2 пин 14 | ❌ |

**5 точек пайки.** CS — просто провод мама-мама в P3.

## Режимы работы

1. **IDLE** — инструкция, кнопка START SCAN, индикатор MCP2515 (зелёный/красный)
2. **SCANNING** — 30 секунд сбора CAN ID, прогресс-бар, пульсирующая точка
3. **LIST** — ID отсортированы по changes, пагинация, тап по ID → монитор
4. **MONITOR** — бар слева, % + raw val справа, без мерцания

## Поиск ID

1. Включить зажигание
2. START SCAN → крутить колёсико яркости 30 секунд
3. В LIST — наверху ID с наибольшим числом изменений
4. Тап → проверка шкалы в MONITOR
5. Не тот? BACK → следующий

## Требования

- ESP32-2432S028 (CYD)
- MCP2515 + TJA1050
- Библиотеки: TFT_eSPI-CYD, MCP_CAN
- Плата: ESP32 Dev Module
