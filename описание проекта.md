# Проект: CAN Brightness Hunter для ESP32 (CYD)

**Версия: 1.0**

## Описание

Прошивка превращает плату **ESP32-2432S028 (Cheap Yellow Display)** + MCP2515+TJA1050 в CAN-сниффер для поиска ID, управляющего яркостью подсветки приборки автомобиля. 

Задача: найти CAN ID диммирования подсветки на Honda Accord 8 (2008-2012), который отсутствует в открытых источниках (opendbc/commaai). После нахождения — управлять дополнительной LED-лентой через MOSFET.

**Аппаратная часть:**
* ESP32-2432S028 (CYD) — основной контроллер + дисплей 320×240
* MCP2515 + TJA1050 — CAN-контроллер (SPI)
* IRLZ44N — N-канальный MOSFET (логический уровень) для LED-ленты
* LED-лента 12V — дополнительная подсветка ног

## Логика работы

### 4 режима работы:

**1. IDLE** — экран ожидания, инструкция, кнопка START SCAN

**2. SCANNING** — сбор данных с CAN-шины в течение 30 секунд:
- Собирает все ID на шине
- Считает **count** (сколько раз ID пришёл)
- Считает **changes** (сколько раз менялись данные в нём)
- Никакой отрисовки пакетов — только таймер + прогресс-бар
- Кнопка STOP для досрочного завершения

**3. LIST** — список найденных ID, отсортированный по changes:
- Самые активные (много изменений) — наверху
- 5 ID на страницу, пагинация кнопками < >
- Тап по ID → открывает MONITOR

**4. MONITOR** — слежение за выбранным ID:
- Первый байт сообщения интерпретируется как яркость (0-100%)
- Большой индикатор с заливкой
- LED-лента управляется в реальном времени
- BACK → возврат в список

### Поиск ID подсветки:
1. Включить зажигание (ON)
2. START SCAN на CYD
3. 30 секунд крутить колёсико яркости
4. В списке — сверху ID с наибольшим changes
5. Тап по первому → проверка реакции
6. Если не тот → BACK → следующий

## Подключение

### MCP2515 → CYD:
| MCP2515 | ESP32 CYD |
|---------|-----------|
| VCC     | 5V |
| GND     | GND |
| SCK     | GPIO18 |
| MOSI    | GPIO23 |
| MISO    | GPIO19 |
| CS      | GPIO32 |

### CAN шина авто → OBD2:
- CAN_H → пин 6
- CAN_L → пин 14
- GND → пин 4
- +12V → пин 16 (через стабилизатор 5V)

### LED-лента:
- GPIO26 → 100Ω → Gate IRLZ44N
- Drain → LED-лента (-)
- Source → GND
- Gate → 10kΩ → GND (pull-down)

## Безопасность
⚠️ **Только чтение!** Скетч не отправляет CAN-сообщения
CAN-шина — ABS, SRS (подушки), двигатель
